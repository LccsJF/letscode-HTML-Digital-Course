Tag negrito

Tag sublinhado=underline

Tag "u"

A tag "u" deixa a palavra ou texto sublinhado, porém por questões de boas práticas e semântica, essa tag caiu em desuso a partir da versão 5 do HTML, sendo o correto utilizar uma tag "span" com o atributo "class=underline" e estilizar no CSS, com estilo text-decoration, opção underline. Tag pré-formatar=pre

Tag "pre" - block=bloco

A tag "pre" deixa um bloco pré-formatado, como se fosse um código.

            nome = claudio
            print(hello, World!)

Tag código=code

Tag "code" - inline=não quebra linha

A tag "code" destaca uma linha de código dentro de um texto.

            nome = claudio
            print(hello, World!)

Como você pode ver na linha acima, a variável nome é do tipo string.


Tag citação=blockquote

Tag "citação=blockquote"

A tag "citação=blockquote" destaca uma linha ou texto em forma de citação.
