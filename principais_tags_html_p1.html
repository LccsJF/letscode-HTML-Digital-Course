Tag Links

Tag para inserir um link de uma pagina ou imagem no html.

(a) = anchor/âncora"
(href=) = hyperlink reference/referência de hiperlink)
("") = texto que descreve a imagem do link

Para abrir link de página em nova aba, utilize a tag: (target="_blank")
