Principais Tags HTML - Parte 2

Tag Image

Tag para inserir uma imagem no html.

(img) = "image/imagem";
(src="") = "source/fonte" - texto que descreve link da imagem;
(alt="") = "alternative/alternativo" - texto que descreve o que é a imagem;




Tag Listas

Lista Não Numerada "ul - unorder list"

  • Primeiro item
  • Segundo item
  • Terceiro item

Lista Numerada "ol - order list"

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
